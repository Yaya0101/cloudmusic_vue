### 注意
    依赖 nodejs版网易云音乐API服务 https://github.com/Binaryify/NeteaseCloudMusicApi

### 基本使用
    克隆源代码后先运行一下 npm i 否则程序无法打包运行
    只完成了基本功能,内容简陋,由于技术能力比较差导致冗余代码比较多
